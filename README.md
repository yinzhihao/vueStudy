# vueStudy

# 创建项目

需要先切换到node版本为v18.16.0，v12的版本安装报错
nvm use 18.16.0

## 方式一

npm init vue@latest

cd runoob-vue3-test
npm install
npm run format
npm run dev

## 方式二

npm init vite-app runoob-vue3-test2

cd runoob-vue3-test2
cnpm install
cnpm run dev

## 方式三

查看是否安装vue cli
vue -V
安装vue CLI
npm install -g @vue/cli

vue create runoob-vue3-app
cd runoob-vue3-app
npm run serve

## 方式四

界面化
vue ui
